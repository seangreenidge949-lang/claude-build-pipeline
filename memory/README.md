# 种子知识（Starter Memory）

这些文件包含使用 Build Pipeline 过程中积累的通用经验。安装时仅在目标文件不存在时复制，不会覆盖你已有的经验记录。

## 文件说明

- `research/search-strategy.md` — 搜索工具优先级和平台选择策略
- `design/pencil-tips.md` — Pencil 设计工具的常见坑点
- `plan/product-doc-lessons.md` — 产品文档评审高频问题
- `review/user-perspective-validation.md` — 验收时的用户视角检查清单
- `implement/INDEX.md` — 技术栈错题本的格式模板

## 如何积累自己的经验

使用 Build Pipeline 过程中，Claude 会自动将踩坑经验写入对应目录。你也可以手动添加。

格式参考 `implement/INDEX.md`。
