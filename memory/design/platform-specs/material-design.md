# Material Design 3 关键规范摘要

## 布局
- 基础网格：4dp
- 标准内边距：16dp（移动端）
- 屏幕参考：360×800dp

## 导航
- Navigation Bar（底部）：80dp 高，最多 5 项
- Navigation Rail（侧边，平板）：80dp 宽
- Top App Bar：64dp（Small）/ 112dp（Medium）

## 触控
- 最小触控区域：48×48dp
- 推荐按钮高度：40dp（Filled）/ 40dp（Outlined）

## 字体
- 推荐：Roboto（默认）/ Google Fonts
- Type Scale：Display / Headline / Title / Body / Label

## 色彩
- Color System：Primary/Secondary/Tertiary + On-colors + Surface/Background
- 支持 Dynamic Color（Material You）

## 组件
- Card 圆角：12dp
- Dialog 圆角：28dp
- Button 圆角：Full（完全圆角）
